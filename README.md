**Висновок**

У ході виконання лабораторної роботи реалізовано об'єктно-орієнтований багатопотоковий застосунок для розв'язання систем лінійних рівнянь методом Якобі під платформу .NET. Реалізовано базову (однопотокову) версію для порівняння та багатопотокову з явним створенням потоків через Thread та синхронізацією через Barrier. Зі збільшенням кількості потоків прискорення зростає не лінійно через: синхронізаційні витрати на бар'єрах, накладні витрати на створення/завершення потоків, false sharing при доступі до спільного масиву next[], обмеження пропускної здатності пам'яті.
